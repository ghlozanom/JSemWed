Agregar elemento a un Container

Para adicionar un elemento a un container antes debe haberse creado un Resource de tipo container.

 

1. De clic derecho sobre el elemento de tipo Container.

 

 

2. Seleccione la opción AgregarElementoContainer.

 

3. En el cuadro de dialogo Agregar un Elemento al Container seleccione el Resource que desea adicionar.

 

 

 

4. De clic en el botón Aceptar.

 

El container tendrá un elemento más.
