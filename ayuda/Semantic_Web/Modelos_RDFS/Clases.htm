Clases

Para poder describir cosas en algún contexto primero se deben poder identificar estas cosas a describir. Estas cosas a describir RDFS las define como clases. Estas clases se pueden utilizar para describir cualquier cosa, como artículos, páginas Web, artículos hasta conceptos abstractos. Las clases son descritas utilizando los recursos de RDFS como rdfs:Class y rdfs:Resource y las propiedades rdf:type y rdfs:subClassOf.

 

En RDFS, una clase es cualquier recurso que tenga como valor de la propiedad rdf:type rdfs:Class. Por ejemplo para decir que ej:Documento es una clase (por convención las clases comienzan por mayúsculas mientras las propiedades por minúscula) se realiza la siguiente aserción:

 

ej:Documento         rdf:type          rdfs:Class

 

rdf:type sirve para decir que un recurso es instancia de una clase determinada. Así, si tenemos:

 

ej:RdfPrimer         rdf:type          ej:Documento

 

Sirve para decir que ej:RdfPrimer es una instancia de la clase ej:Documento. Un recurso puede instanciar más de una clase.
