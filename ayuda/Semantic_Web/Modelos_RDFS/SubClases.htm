SubClases

Para indicar que una clase es una especialización de otra se utiliza rdfs:subClassOf. Por ejemplo:

 

ej:Informe         rdfs:subClassOf         ej:Documento

 

Dice que informe es de tipo ej:Documento. Esto se interpreta como que ej:Informe es un ej:Documento más ej:Documento no necesariamente es un ej:Informe. Sin embargo esta interpretación debe darla el software que debe ser escrito para entender y procesar este tipo de aserciones. La propiedad rdfs:subclaseOf es transitiva, así si tenemos:

 

ej:InformeParcial         rdfs:subClassOf         ej:Informe

 

También será cierta la siguiente aserción:

 

ej:InformeParcial         rdfs:subClassOf         ej:Documento

 

RDFS define todas las clases como subclases de rdfs:Resource.
