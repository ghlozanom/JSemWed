RDF Containers

En algunas ocasiones se necesita describir un grupo de cosas como, por ejemplo, los estudiantes de un curso, o los usuarios de cierto servicio. RDF provee cierto vocabulario que permite definir este tipo de cosas.

 

Primero que todo, RDF define Container, el cual es un Resource que contiene cosas. RDF define tres tipos de Containers:

 

  • rdf:Bag

  • rdf:Seq

  • rdf:Alt

 

rdf:Bag es un container en el cual no importa el orden de las cosas.

 

rdf:Seq es un container en el cual el orden de las cosas es importante.

 

rdf:Alt es un container en el cual las cosas representan alternativas. Por ejemplo en un container tipo rdf:Alt se pueden describir los sitios de donde se puede descargar una librería de código.
