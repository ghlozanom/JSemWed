Ingresar Resources

Un sistema que permita identificar de manera inequívoca las diferentes partes que componen las aserciones ya esta disponible en la Web: Uniform Resource Identifier (URI). Las URL que identifican las páginas Web son un tipo de URIs. Las URIs en Semantic Web sirven para identificar todo tipo de recursos, no solamente los asequibles en la red. Los tipos de recursos que se pueden identificar con los URI son:

 

  •  Cosas asequibles en la red (como páginas Web o documentos electrónicos).

  • Cosas no asequibles en la red. (como por ejemplo personas, libros, árboles, etc.).

  • Conceptos abstractos (como creador, director, etc.).

 

Las URI permiten identificar las partes de las aserciones de manera única. Permiten distinguir los Predicados y permite tratar estos Predicados como recursos en sí. Si, por ejemplo, en un contexto se utiliza un Predicado nombre para identificar el nombre de un ser humano y en otro contexto se utiliza el mismo Predicado nombre para  el nombre que identifica las variables en un algoritmo, ¿Cómo haría una aplicación para distinguir estos dos Predicados? Y además, al identificar los Predicados con un URI estos predicados se pueden utilizar como Sujetos de otras aserciones.

 

Estos URI son compatibles con el sistema UNICODE que permite utilizar caracteres de diferentes lenguajes.
