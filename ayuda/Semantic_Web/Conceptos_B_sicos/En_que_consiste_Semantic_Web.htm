En que consiste Semantic Web

La definición más clara, define a Semantic Web como una colección de conocimientos que evoluciona, construida con una arquitectura que no solo permite a los humanos intercambiar este conocimiento, sino que también a programas y agentes de software inteligentes que sean capaces de entender este conocimiento, para utilizarlo y para añadir conocimiento útil a los otros actores.

 

Semantic Web se basa en un concepto que se conoce como marcos conceptuales, los cuales constituyen una ciencia relacionada con la representación del conocimiento. La W3C ha desarrollado un estándar llamado RDF, el cual, al igual que en las proposiciones de los marcos conceptuales, consta de 3 secciones: Sujeto, Predicado y Objeto. Por medio de estas secciones, se crean los Statements o aserciones, las cuales permiten decir cosas (Predicado y Objeto) acerca de recursos (Sujetos). Por ejemplo:

 

William Shakespeare escribió Hamlet.

 

Dice acerca del Sujeto (William Shakespeare) el predicado (escribió) con el valor de Hamlet. Aserciones como esta se pueden decir utilizando RDF. Aparte, también se pueden decir más cosas acerca de William Shakespeare como, por ejemplo, que era de origen Británico. También se puede utilizar el Objeto, que en este caso es Hamlet, como Sujeto de otras aserciones. Por ejemplo, sobre Hamlet se puede decir que es una obra de teatro o que pertenece a la literatura inglesa.
